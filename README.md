# Abbot Study

Abbot Study is an AI-native education operating system designed to guide learners through ordered, mastery-based learning using trusted curriculum content, The Abbot tutor, Ariel teach-back, adaptive assessment, remediation, and learning intelligence.

## Current Build Phase

Phase 1A — Project Skeleton

## Core Principle

Learning order must never depend on async processing order, database IDs, timestamps, queue completion order, or uncontrolled AI decisions.
